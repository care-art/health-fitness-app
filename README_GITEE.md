# 健康健身助手

免费在线健康健身计算器，提供BMI指数、体脂率、基础代谢率、每日热量消耗、腰臀比、运动卡路里消耗、每日饮水量计算，以及个性化营养计划生成。

## 功能特性

- 📊 BMI 计算器
- 🔥 体脂率计算
- ⚡ 基础代谢率 (BMR)
- 🍎 每日热量消耗 (TDEE)
- 📏 腰臀比计算
- 🏃 运动卡路里消耗
- 💧 每日饮水量建议
- 🥗 个性化营养计划

## 技术栈

- React + TypeScript
- Tailwind CSS
- Vite
